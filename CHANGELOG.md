# 📜 История версий (Changelog)

Все примечательные изменения для этого проекта будут задокументированы в этом файле.

## [1.0.0] - 2026-04-17

### ✨ Добавлено

#### Основной функционал
- ✅ Полная поддержка Manifest V3
- ✅ Service Worker для обработки API запросов к OpenRouter
- ✅ Content Script для сканирования и подчеркивания ошибок
- ✅ Проверка текста в реальном времени с дебаунсом (600ms)
- ✅ Волнистое подчеркивание ошибок (красное и желтое)
- ✅ Всплывающее окно с вариантами исправления
- ✅ Кнопки "Исправить это" и "Исправить всё"

#### Поддержка моделей
- ✅ Google Gemini 2.0 Flash (`:free`)
- ✅ Meta Llama 3.3 70B Instruct (`:free`)
- ✅ DeepSeek R1 (`:free`)
- ✅ Mistral AI 7B (`:free`)
- ✅ Alibaba Qwen 2 7B (`:free`)
- ✅ Автоматическое обновление списка доступных моделей
- ✅ Фильтрация только бесплатных моделей (`:free`)

#### Настройки
- ✅ Страница опций (Options) с полной конфигурацией
- ✅ 3 уровня глубины проверки (орфография, грамматика, полный анализ)
- ✅ Выбор жанра (General, Academic, Creative)
- ✅ Поддержка пользовательских промптов
- ✅ Белый список сайтов для исключения
- ✅ Исключаемые правила проверки
- ✅ Сохранение настроек через `chrome.storage.sync`

#### Интерфейс
- ✅ Material Design 3 оформление
- ✅ Темная тема (автоматическая смена)
- ✅ Адаптивность (мобильные устройства)
- ✅ Плавные анимации (`cubic-bezier`)
- ✅ Быстрое меню (Popup) расширения
- ✅ Индикатор загрузки (пульсирующий кружок)
- ✅ Указатель статуса расширения (зеленый/красный)

#### Безопасность
- ✅ API ключ хранится только локально в `chrome.storage`
- ✅ Никаких внешних серверов-прокси
- ✅ Прямые запросы к OpenRouter API
- ✅ Валидация JSON ответов
- ✅ Обработка ошибок и fallback механизмы

#### Документация
- ✅ README.md с полным руководством
- ✅ INSTALLATION.md с пошаговыми инструкциями
- ✅ ARCHITECTURE.md с описанием архитектуры
- ✅ EXAMPLES.md с примерами и решением ошибок
- ✅ Комментарии на русском языке во всем коде

### 🎠 Изменено

- Обновлены иконки SVG для лучшей масштабируемости
- Улучшена чувствительность парсера JSON (добавлены fallback'и)
- Оптимизирована обработка больших текстов (>3000 символов)

### 🐛 Исправлено

- Исправлена ошибка с дублированием всплывающих окон
- Исправлена обработка contenteditable элементов
- Исправлена ошибка при повторной отправке текста

### 🚀 Производительность

- Дебаунс (600ms) для минимизации API запросов
- Кэширование результатов проверки в памяти
- Эффективная работа с mutationobserver
- Ограничение размера текста (макс 4000 символов)

## Документ по семантическому версионированию

Этот проект следует [Semantic Versioning](https://semver.org/):

- **MAJOR** (1.0.0) = Несовместимые изменения API
- **MINOR** (1.1.0) = Новые функции, обратно совместимые
- **PATCH** (1.0.1) = Исправления ошибок, обратно совместимые
