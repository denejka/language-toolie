# 🚀 Быстрая настройка расширения

Этот файл содержит минимально необходимые шаги для быстрого запуска расширения.

## 30 секунд до работающего расширения

### Шаг 1: Указать API ключ (1 минута)

1. Перейти на https://openrouter.io
2. Зарегистрироваться (Google/GitHub/Email)
3. Скопировать API ключ из Settings → API Keys

### Шаг 2: Загрузить в Chrome (1 минута)

1. Открыть `chrome://extensions/`
2. Включить "Режим разработчика" (вверху справа)
3. Нажать "Загрузить распакованное расширение"
4. Выбрать папку `/language-toolie`

### Шаг 3: Настроить (30 секунд)

1. Нажать на иконку расширения
2. Нажать "⚙️ Настройки"
3. Вставить API ключ
4. Нажать "Проверить соединение" → должна быть зеленая галочка
5. Готово! ✅

## Что дальше?

- 🎯 Перейти на любой сайт с текстовым полем
- ✍️ Начать писать текст с ошибкой
- 👀 Видеть красное волнистое подчеркивание
- 🖱️ Кликнуть на ошибку → выбрать исправление
- 🎉 Готово!

## Пример текста с ошибкой

Скопируйте и вставьте в любое поле:

```
Этот расш иренне содержит множество оишбок в орфографи
```

Если расширение работает, все слова будут подчеркнуты.

## Проблемы?

**Расширение не работает:**
1. Убедиться что оно включено (иконка видна в тулбаре)
2. Попробовать перезагрузить расширение (кнопка на странице с расширениями)
3. Очистить кэш (меню расширения → 🗑 Чистка)

**Ошибка соединения:**
1. Проверить API ключ (не должно быть пробелов)
2. Убедиться что ключ начинается с `sk-or-`
3. Проверить интернет соединение
4. Попробовать создать новый ключ на openrouter.io

**Нет моделей в списке:**
1. Убедиться что API ключ правильный
2. Нажать кнопку "🔄 Обновить список моделей"
3. Или выбрать одну из встроенных моделей

## Первое использование - лучшие практики

1. **Начните с простого текста** (2-3 предложения)
2. **Установите уровень 2** (стандартный - хороший баланс)
3. **Используйте модель Gemini** (самая быстрая)
4. **Со временем экспериментируйте** с другими моделями

## Оптимальные настройки для разных целей

### Для быстрой проверки
- Модель: Google Gemini 2.0
- Уровень: 1 (орфография)
- Жанр: General

### Для работы с почтой
- Модель: Llama 3.3 70B
- Уровень: 2-3 (полный)
- Жанр: Academic

### Для творчества
- Модель: DeepSeek R1
- Уровень: 3 (полный)
- Жанр: Creative

## Советы

💡 **Совет 1:** API ключ хранится только в вашем браузере. Никому не давайте его.

💡 **Совет 2:** Больше используя бесплатные модели - экономьте деньги!

💡 **Совет 3:** Если расширение работает медленно, попробуйте модель Gemini (она быстрее).

💡 **Совет 4:** Вы можете выключить расширение на конкретных сайтах.

💡 **Совет 5:** Темная тема включается автоматически если у вас темный режим в браузере.
