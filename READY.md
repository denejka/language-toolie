🚀 БЫСТРЫЙ СТАРТ (3 МИНУТЫ)
────────────────────────────────────────────────────────────────────────────────

1️⃣ ПОЛУЧИТЬ API КЛЮЧ (1 минута)
   → Перейти на openrouter.io
   → Зарегистрироваться
   → Скопировать API ключ

2️⃣ ЗАГРУЗИТЬ РАСШИРЕНИЕ (1 минута)
   → Открыть chrome://extensions/
   → Включить "Режим разработчика"
   → "Загрузить распакованное расширение"
   → Выбрать папку language-toolie

3️⃣ НАСТРОИТЬ (1 минута)
   → Нажать на иконку расширения
   → Нажать "⚙️ Настройки"
   → Вставить API ключ
   → "Проверить соединение" ✅

Готово! Расширение начит работать сразу же!

🔐 БЕЗОПАСНОСТЬ
────────────────────────────────────────────────────────────────────────────────

✅ API ключ хранится только локально
   • chrome.storage.sync (зашифровано)
   • Никогда не отправляется на внешние серверы
   • Доступен только Service Worker'у

✅ Текст отправляется только в OpenRouter API
   • Прямое HTTPS соединение
   • Никаких прокси-серверов
   • Полная конфиденциальность

✅ Локальное кэширование
   • Результаты сохраняются в памяти браузера
   • Нет загрузки на облако
   • Контроль пользователя над данными
