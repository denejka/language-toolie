# AI Grammar & Style Checker

Современное расширение для Chrome с использованием нейросетевых моделей через OpenRouter API для проверки грамматики и стилистики текста.

## 🎯 Особенности

✅ **Проверка в реальном времени** - анализ текста с дебаунсом 600ms  
✅ **Волнистое подчеркивание ошибок** - аналог LanguageTool  
✅ **Всплывающее окно с исправлениями** - Material Design 3 интерфейс  
✅ **Нейросетевые модели** - использует бесплатные модели OpenRouter API  
✅ **Гибкие настройки** - 3 уровня глубины проверки  
✅ **Безопасность** - API ключ хранится только локально в браузере  
✅ **Дизайн** - поддержка темной темы и мобильной адаптивности  

## 🚀 Быстрый старт

### 1. Получение API ключа

1. Перейти на [OpenRouter.io](https://openrouter.io)
2. Зарегистрироваться или войти
3. Получить API ключ из личного кабинета

### 2. Установка расширения

1. Открыть Chrome и перейти на `chrome://extensions/`
2. Включить "Режим разработчика" (верхний правый угол)
3. Нажать "Загрузить распакованное расширение"
4. Выбрать папку с файлами расширения

### 3. Первоначальная настройка

1. Нажать на иконку расширения в тулбаре Chrome
2. Нажать "⚙️ Настройки"
3. Вставить API ключ OpenRouter
4. Выбрать модель из списка
5. Нажать "Проверить соединение"

## ⚙️ Использование

**Уровни глубины проверки:**
- 1️⃣ Только орфография
- 2️⃣ Грамматика и пунктуация
- 3️⃣ Полный анализ + логика

**Доступные модели:**
- Google Gemini 2.0 Flash (быстрая)
- Llama 3.3 70B (мощная)
- DeepSeek R1 (точная)
- Mistral 7B (легкая)
- Qwen 2 7B (многоязычная)

## 🔒 Безопасность

- ✅ API ключ хранится только локально
- ✅ Все запросы идут напрямую к OpenRouter API
- ✅ Данные кэшируются локально в браузере
